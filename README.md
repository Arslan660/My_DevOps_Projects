# My DevOps Projects

A collection of hands-on DevOps projects demonstrating containerization, CI/CD automation, and Infrastructure as Code on AWS. Built as part of my journey into DevOps engineering.

## 📁 Projects

| Project | Description | Status |
|---|---|---|
| [docker-node-app](./projects/docker-node-app) | Node.js app containerized with Docker (multi-stage builds, volumes, networking) | ✅ Completed |
| [ci-cd-pipeline](./projects/ci-cd-pipeline) | Automated CI/CD pipeline using GitHub Actions to build, test, and deploy a Dockerized app | ✅ Completed |
| [terraform-aws-infra](./projects/terraform-aws-infra) | AWS infrastructure (VPC, subnet, EC2) provisioned entirely with Terraform | ✅ Completed |

Each project has its own README with setup instructions, architecture notes, and key learnings.

## 🛠️ Tech Stack Across Projects
- **Containers:** Docker, Docker Compose
- **CI/CD:** GitHub Actions
- **Cloud:** AWS (VPC, EC2, Security Groups)
- **IaC:** Terraform
- **Languages:** JavaScript (Node.js), HCL, YAML
